웹 사이트 디자인을 담당하는 언어, CSS


Html은 정보를 표현하기 위해 만들어진 언어인데, 사람들은 정보만으로는 못산다고 불평을 쏟아내기 시작합니다.

바로 이런 투박한 디자인이기 때문이죠.



Tag? Style? Language?


디자인을 표현하기 위해 사람들은 Html에 새로운 태그를 만들지, 아예 디자인만을 위한 새로운 언어를 만들기 고민합니다.



CSS의 등장


html에 새로운 태그를 추가해 써보았지만 한계에 부딫혀 한 언어가 탄생하게 되었습니다.


바로 CSS가 개발된 이후, 우리는 드디어 아래처럼 현대적인 레이아웃을 갖춘 웹사이트들을 만나볼 수 있게 되었습니다.



CSS의 정의


CSS는 Cascading Style Sheet의 약자로, 문서의 콘텐츠(정보)와 레이아웃 글꼴 및 시작적 요소들로 표현되는 문서의 외관(디자인) 인)을 분리하기 위한 목적으로 만들어졌습니다.


CSS가 혁명적인 이유는 내부 스타일, 외부 스타일을 따로 지정하여 가변적인 코딩이 가능해졌고 스타일 시트를 생성해 다른 사람에게 공유하여 손쉽게 다른 사람이 만든 디자인을 적용할 수 있게 된 것입니다.

또한, 이 덕분에 우리는 스타일 css 파일의 코드 한줄만 고쳐주면 전체 html 파일의 디자인을 바꿀 수 있게 되었습니다.
